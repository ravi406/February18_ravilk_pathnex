## Ansible Task - install Nginx on Pathnex Server

rewrite this YAML Manually:

---
- name: install Nginx on Pathnex Server
  hosts:all
  become:Yes

  tasks:
    - name: install nginx
       yum:

        name:nginx
        state:present

## 🔹Kubernetes Task - create Nginx Pod
api version:v1
kind:Pod
metadata:
    name:pathnex-pod
    spec:
    containers:
    -name:web
    image:nginx:latest
    ports:
        -containerPort:80
